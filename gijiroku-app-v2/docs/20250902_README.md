## マスターファイル

## 開発記録の付け方

1. 新しいセッションを開始する際は、`sessions/SESSION_LOG_TEMPLATE.md` をコピーして日付付きファイルを作成
2. 作業内容を記録
3. 重要な変更は `development/DEVELOPMENT_LOG.md` に集約
4. アーキテクチャの変更は該当ドキュメントを更新
